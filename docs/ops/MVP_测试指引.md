## 核心测试路径（5 条）

### 路径 1：数据源管理
登录 admin → 连接中心 → 新建数据源 → 测试连接 → 保存

### 路径 2：DDL 合规检查
连接中心 → 选择数据源 → 触发扫描 → 查看违规报告

### 路径 3：AI 问答
首页 → 选择数据源 → 输入"你有几个数据源" → 查看流式回答

### 路径 4：Tableau 资产浏览
资产管理 → Tableau 资产列表 → 点击资产详情 → 查看健康分

### 路径 5：权限隔离
用 user1 登录 → 尝试访问 /admin/users → 应被拒绝（403 或重定向）
